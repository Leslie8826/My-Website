Oracle
  • We are given an excel file, HR_data.xlsx, with a lot of data on a company's employees. The purpose of this project is to analyse the data according to some requirements.

  • After analysing the excel file and after some brainstorming, we came up with a list of categories based upon the type of data. Each category would represent a table in our database. We wrote everything on paper.

  • We draw each table on paper and added the relationships between tables when necessary, which formed our Database model. Then we draw our data model in Oracle Data Modeler.


  • We created our database with ddl statements in Oracle SQL Developer from the database model built in point 2.

  • We also created a stage table where we inserted all the data that exist from the excel file HR_Data.xlsx.

  • We populated the database with dml statements. The data inserted came from the stage table. The table created are below:
